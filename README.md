# MyPromptPro

Astro static site. Migrated from plain HTML.

## Structure

- `src/layouts/BaseLayout.astro` — shared layout (head, fonts, base styles)
- `src/pages/index.astro` — home/landing page
- `src/pages/teachers/index.astro` — Teachers prompt pack product page
- `public/assets/` — static assets (images)

## Development

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # output to dist/
npm run preview  # preview the build
```
